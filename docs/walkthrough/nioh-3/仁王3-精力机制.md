# 仁王3 精力机制 ⚡

精力机制是仁王系列最核心的战斗系统，掌握精力机制是制胜的关键。本文将深入解析精力系统的运作原理，帮助你建立正确的对敌思路。

## 🎯 基础认知

### 精力条识别

- **位置**：精力条显示在敌人血条下方
- **颜色区分**：
  - 🟣 **紫色** = 妖怪
  - 🔵 **蓝色** = 人类

::: tip 快速判断
通过精力条颜色可以快速判断敌人类型，从而采用不同的战斗策略。
:::

## 👥 敌人分类与精力机制

根据精力机制的不同，敌人分为两大类：人类和妖怪。

### 🧑 人类精力机制

人类的精力系统与玩家类似，遵循消耗-恢复的循环机制。

#### 精力特性

1. **自然恢复**：常态下快速恢复
2. **防御影响**：防御时减慢或停止恢复
3. **消耗方式**：
   - 被攻击命中会减少
   - 主动使用招式会消耗

#### 处决机制

当人类精力被削减至 0 时：
- 陷入硬直状态
- 锁定标志变为 🔴 **红色**
- 可以进行处决

#### 防御状态差异

人类有两种状态，精力削减计算方式不同：

| 状态 | 精力削减计算 |
|------|------------|
| 非防御状态 | 武技精力伤害 |
| 防御状态 | 武技崩解伤害 |

::: tip 战斗建议
- 对防御型敌人使用崩解伤害高的武技
- 对进攻型敌人直接输出精力伤害
:::

### 👹 妖怪精力机制

妖怪的精力系统更加复杂，具有独特的机制。

#### 精力特性

1. **不自然恢复**：精力不会自动恢复
2. **环境恢复**：仅在常世或常暗中缓慢恢复
3. **低消耗**：几乎不消耗精力，只有强力攻击会消耗
4. **双层机制**：
   - 🔵 **蓝色部分**：当前精力值
   - 🔴 **红色部分**：精力上限

#### 精力上限机制

妖怪独有的精力上限系统：

```
[██████      ] 蓝色精力
[         ███] 红色上限
```

- **上限削减**：持续攻击削减红色上限
- **上限清空**：红色部分清空后可以处决
- **精力清空**：蓝色部分清空后失去霸体

#### 战斗策略

1. **优先削减上限**：使用守护灵技快速削减红色上限
2. **破除霸体**：精力清空后妖怪失去霸体，可连续输出
3. **处决时机**：上限清空后立即处决

::: warning 重要提示
守护灵技对妖怪的上限削减能力极强，精力清空后建议：
1. 先释放守护灵技削减上限
2. 再进行连续输出
3. 上限清空后立即处决
:::

## 💡 实战技巧总结

### 对人类

- ✅ 持续施压，不给恢复时间
- ✅ 针对防御状态选择合适武技
- ✅ 精力归零立即处决

### 对妖怪

- ✅ 优先削减精力上限（红色部分）
- ✅ 善用守护灵技
- ✅ 精力清空后连续输出
- ✅ 避免在常世/常暗中长期战斗

::: tip 进阶建议
熟练掌握精力机制后，可以通过观察敌人精力条来预判攻击时机，实现无伤战斗。
:::
