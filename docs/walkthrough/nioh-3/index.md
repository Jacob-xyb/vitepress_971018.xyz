# 仁王3 攻略指南

## 核心系统介绍

### 武士系统

#### 琢技机制

琢技是武士系统的核心机制，发动期间可获得以下增益：

- **强攻击和武技得到强化**
- **琢技攻击不消耗精力**
- **使用不同武技可持续维持琢技状态**

#### 无限琢技连招

通过合理的技能衔接，可以实现琢技的无限循环：

**核心要点：**

1. 琢技可通过不同的强攻击和武技延长持续时间
2. 流转或残心后可派生 `冬月`（武士右侧派生界面）
3. `冬月` 本身也算武技，可延长琢技
4. 精力消耗通过流转2回复

**最简连招：**

```
冬月 → 武技 → 冬月 → 循环
```

::: tip 推荐流派
并非所有流派都适合无限琢技，需考虑流转和武技的衔接流畅度。

目前最推荐：[太刀流派](./武士太刀)
:::

## 里技系统

::: warning 注意事项
里技不适宜长期使用，容易损坏手柄且操作疲劳。
:::

### PS5 推荐设置

为了更好地使用里技，建议调整以下按键：

1. **跳跃改为 ○**，**闪避改为 ×**
2. **奔跑改为长按闪避键**

### 基础里技详解

#### 里技1：冲A

**操作方式：** 奔跑中 + 轻攻击

**优势：**
- 精力消耗低于普通平A
- 攻击后可立即接派生技能

#### 里技2：闪A

**操作方式：** 闪避中 + 轻攻击

**优势：**
- 闪避后可以马上接平A
- 精力消耗低于普通平A
- 攻击后可立即接派生技能

#### 里技3: 流转2

**操作方式：** 残心时按住R1 + 不同流派1 + 不同流派2

流转2可以在精力不足时使用

#### 里技4：纳闪A

::: tip 专属技能
太刀流派专用技能
