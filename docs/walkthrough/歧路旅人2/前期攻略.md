# 前期攻略

## 必拿道具

### JP增幅器

装备效果：提升JP获取量（1.5倍）

地点：托托哈哈岛 - 海鸣洞穴

装有『JP增幅器』的宝箱位于存档点的右上方，需要先往左边上楼梯，然后到岔路口再一直往右走才能到达。

### 繁盛护身符

装备效果：战斗结束时获得的金额增加（约1.1倍）

『繁盛护身符』可以通过完成支线任务「醉醺醺的贵族」奖励获得。此支线位于「利夫兰多-库罗普提尔」

将时间段切换至夜晚，进入城镇上的酒馆与「酒馆的女孩」交谈开启任务。

对「醉醺醺的贵族」使用地图指令使其晕厥（偷袭、进入梦乡等）。

之后看完剧情即可完成任务并获得4000叶币以及「繁盛护身符」。

### 召唤胜利的戒指

装备效果：物理攻击+20、装备效果【战斗开始时赋予物攻加成的效果】

『召唤胜利的戒指』可以通过完成支线任务「流连于赌场的男子」奖励获得。此支线触发位置在「布兰德兰多地区-新德尔斯塔」。

1. 与「新德尔斯塔-小道-」入口附近的「担心丈夫的妻子」对话。
2. 前往「新德尔斯塔-小道-」北边的赌场。
3. 对赌场内的「庄家」使用地图指令获取情报「赌场的真相」（探查、收买等）。
4. 返回小道向「担心丈夫的妻子」报告即可完成任务并获得「恢复BP的石榴」和「召唤胜利的戒指」。

### 凯特林的天木蓼

装备效果：更容易遭遇凯特林

托托哈哈岛 - 猫奴妇人 - 晕倒后进屋获得

### 黑市-大精灵石

黑市会在新德尔斯塔平原的晚上开启

切换昼夜可以更改商人类型，可以不断切换昼夜直到出现出售想要购买商品的商人。

各属性精灵石（大）在应对各种强力怪物或稀有怪时非常有用，伤害大约在2000左右，价格只有1700，性价比非常高。尤其是能够一击击败稀有怪凯特林和奥克托林。

> 💡 **Tips**：卖精灵石的是两个女神官，非常难刷新出来，所以一次可以多买一点

获取方法：

2. 遇怪撒石头，猎人抓取
