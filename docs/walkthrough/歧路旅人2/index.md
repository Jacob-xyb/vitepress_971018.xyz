# 歧路旅人2

> 《歧路旅人2》（Octopath Traveler 2 / オクトパストラベラー2）—— Square Enix 制作的 HD-2D 风格 JRPG，本作舞台位于「索里斯蒂亚」大陆，讲述 8 位来自不同背景的旅人各自的旅途。

## 🌍 索里斯蒂亚（Solistia）大陆

本作舞台分为东西两块大陆与多个海上岛屿：

- **西部**：寒霜之国（Ku）、神圣教国（Flamesgrace）、学术之都（Solistia）
- **东部**：海港城市、工业小城（Montblanc）、宁静乡村（Cropdale）
- **岛屿**：边境之岛（Toto'haha）等
