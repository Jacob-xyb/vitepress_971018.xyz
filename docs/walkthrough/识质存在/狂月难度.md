# 识质存在 狂月难度攻略

## 难度特点

- 敌人伤害和攻击速度提升
- 敌人血量明显增加
- 只能在一周目进行（无法携带升级物品）
- 打印机可提前解锁部分内容

::: warning
狂月难度是游戏的最高难度，建议完成一周目并充分准备后再挑战。
:::

## 战前准备

### 资源准备

| 资源 | 建议数量 | 用途 |
|------|----------|------|
| 金币 | 1-2 万 | 强化月炮 |
| 月矿 | 若干 | 武器强化 |
| 升级组件 | 若干 | 固件升级 |

### 固件升级优先级

1. **工作服（生命/防御）** → 5 级
2. **侵入（骇入伤害）** → 5 级
3. **主要装置（武器伤害）** → 3-5 级

## 通用战斗技巧

### 敌人类型应对

| 敌人类型 | 应对策略 |
|----------|----------|
| 杂兵群 | 蓝绿武器拖延，或群体+混乱处理 |
| 精英怪 | 降防 + 必杀 + 喷子/充能枪速杀 |
| BOSS | 破盾 → 骇入 → 输出 循环 |

### 核心战斗循环

```
破盾 → 骇入破甲 → 连接增幅节点 → 月炮蓄满 → 爆发输出
```

---

## 最终 BOSS 打法

### 阶段一：磨血至半血

1. 开场 **L1 蓄力两次** 破盾
2. 骇入破甲（**不要用骇入节点**）
3. 先破甲再骇入一次，出现增幅节点
4. 把所有增幅节点和骇入节点都连起来
5. BOSS 被定住后月炮蓄满，一发打进半血

::: important
一定要确保半血进过场动画！
:::

### 阶段二：斩杀

动画结束后：
1. 直接开大定住
2. 一边蓄力一边骇入
3. 第二炮应该就死了
4. 没死也会进入过热处决

::: tip
手残党前半段可以用凿岩镐磨血，配合「最后一发几率不消耗」能磨掉不少血。
:::

## 月炮强化详解

### 强化优先级

月炮是最终战的核心武器，优先强化。

### 强化建议

- 存 **1-2 万** 金币用于月炮
- 蓄满瞬间释放 **333 一发**
- 留 **2-3 千** 金币当弹药

### 弹药管理

不要把金币全用来强化，留够弹药钱。

---

## 实用技巧

1. **不要等武器没了再用大招** — 配合主力武器打爆发
2. **推图带大招充能 +15% 的模块** — 有奇效
3. **活用蓄力** — L1 蓄力可以快速破盾
4. **骇入优先破甲** — 破甲后增幅节点伤害更高
5. **善用混乱状态** — 敌人会互相攻击
6. **关注蓝色增伤节点** — 构筑速攻流
