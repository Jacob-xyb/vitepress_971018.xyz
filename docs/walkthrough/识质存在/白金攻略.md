# 《识质存在》白金攻略

## 白金概览

| 项目 | 数据 |
|------|------|
| 奖杯数量 | 36 个（21 铜、9 银、5 金、1 白金） |
| 白金时长 | 约 24-25 小时 |
| 全收集时长 | 约 20 小时 |
| 难度要求 | 需通关狂月难度 |

- 白金周目: 2周目
	- 任意难度一周目
	- 最高难度重新打一周目
- 有无错过杯： 无

---

白金Tips:
1. 训练模式并不需要全部perfect，只需要完成。
2. `未知信号`决战一定要带`黑盒`模块，否则不跳杯
3. `借用护盾`奖杯需要对防卫者，举盾后用`代码生成器`+`混乱`节点即可跳杯

## 白金步骤

本攻略将白金过程分为 **4 个阶段**，按顺序完成即可白金。

---

## 步骤一：一周目流程 ⏱️ 7-10 小时

一周目的目标是：**通关主线 + 顺便收集能收集的物品**

### 游玩建议

- 难度选择任意（简单/普通皆可）
- 跟着主线走，不用刻意刷收集
- 遇到能拿的物品顺手拿一下
- **注意**：有些收集品需要在特定节点才会出现

### 需要注意的奖杯

| 奖杯 | 说明 | 达成时机 |
|------|------|----------|
| 一网打尽 | 用月丝回收器单次击败 3 名敌人 | 主线推进到中后期可做 |
| 别想逃 | 击杀一只清扫者 | 主线中随机遭遇 |
| 借用下，行吗？ | 对敌人使用混乱借用护盾 | 骇入能力解锁后 |

::: tip
这些奖杯在流程中自然遇到，不用刻意刷。如果漏了，二周目补也很快。
:::

---

## 步骤二：开放狂月难度 ⏱️ 1-2 小时

### 解锁条件

- 通关一次任意难度后
- 回防护站与阿舱对话
- 选择「新游戏+」并切换为「**狂月难度**」

### 进入前准备

- 打印机已解锁凿岩镐、月丝加农炮的打印
- 建议带够 **1-2 万金币**用于强化月炮
- 存档备份（以防需要重打）

### 关键系统解锁

| 系统 | 解锁条件 | 作用 |
|------|----------|------|
| 固件升级台 | 主线推进 | 提升生命、防御、骇入能力 |
| 装置打印机 | 主线推进 | 强化武器伤害 |
| 模拟训练 | 区域 2 完成后 | 获取升级资源 |

### 模拟训练（战前资源准备）

模拟训练是获取**阿昌币、纯月矿、月石、升级组件、武器配件**等核心资源的最稳定高效渠道。

::: tip 推荐
前期优先清完所有**主目标**（限时清敌类）。次要目标不影响核心奖励，可随缘补。
:::

**重点关卡**：

| 关卡 | 名称 | 用途 |
|------|------|------|
| 关卡 10 | 捕鼠器 | 「一网打尽」奖杯 |
| 关卡 16 | 暴露节点 | 「谁还需要枪？」奖杯 |
| 关卡 29-30 | 高难挑战 | 资源收益高 |

详细攻略：[模拟训练](./模拟训练)

## 步骤三：狂月难度通关 ⏱️ 10-15 小时

这是白金最难的部分，需要充分准备。

### 核心思路

```
生存 → 破盾 → 骇入 → 输出 → 循环
```

### 武器与模组配置

**推荐武器**：喷子 + 蓄力枪 + 月丝炮

**月炮模组**：

| 模组 | 效果 |
|------|------|
| 诅咒 | 增伤 |
| 强硬进攻 | 伤害提升 |
| 超速充填 | 快速充能 |
| 效能强化 | 效率提升 |
| 热量转移 | 热量管理 |

**骇入节点**：降防、冻结、过热

**模式晶片**：增幅模式

### 战斗流程

1. 开场 L1 蓄力两次破盾
2. 骇入破甲（不用骇入节点）
3. 再骇入一次出现增幅节点
4. 连接所有增幅+骇入节点
5. BOSS 定住后月炮蓄满（一发打半血）
5. **确保半血进过场动画**
6. 动画后开大 + 蓄力 + 骇入
7. 第二炮击杀

详细打法见：[狂月难度](./狂月难度)

## 资源高效获取

### 模拟训练

完成主线后推荐刷模拟训练，是最稳定的资源来源：

| 类型 | 奖励 |
|------|------|
| 限时清怪类 | 阿昌币、月矿、月石、升级组件 |
| 骇入挑战类 | 骇入组件、升级组件 |
| 生存防御类 | 防御配件、月矿 |

### 升级优先级

1. **工作服（生命/防御）** → 5 级保命
2. **侵入（骇入伤害）** → 5 级核心输出
3. **主要装置（武器伤害）** → 3-5 级够用

## 常见问题

### Q：狂月难度死了会怎样？
A：可以随时重新挑战，建议多利用存档点。

### Q：月炮强化需要多少钱？
A：建议准备 1-2 万金币，留 2-3 千当弹药。

### Q：有些奖杯漏了怎么办？
A：二周目可以快速补，建议先打完狂月难度再补。

### Q：地忆芯片全收集难吗？
A：数量较多，建议流程中顺手收集，遗漏最后统一补。
