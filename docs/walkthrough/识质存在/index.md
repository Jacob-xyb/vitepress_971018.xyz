# 识质存在 攻略指南

## 游戏简介

《识质存在》(PRAGMATA) 是卡普空耗时六年打造的科幻动作冒险游戏。玩家将跟随命运多舛的调查队队员"休"和机器人小女孩"戴安娜"，在被失控 AI 掌管的月球设施内寻找返回地球的方法。

游戏融合了第三人称射击与独特的"侵入"解谜系统，需要同时操控休进行射击战斗、操作戴安娜进行黑客解密。游戏时长约 10 小时，白金约需 24-25 小时。

## 游戏设定

### 核心机制

**1. 双角色系统**

玩家同时操控两名角色：
- **休**：进行移动、射击、跳跃等物理操作
- **戴安娜**：在肩部进行侵入（骇入）操作

**2. 侵入系统**

戴安娜的侵入是一个点击/划线类小游戏，需要在 3×3 至 5×5 网格中规划路径破解敌人防御。节点类型：
- 🔵 **蓝色节点** — 增伤节点
- 🟡 **黄色节点** — 解码器节点
- 🔴 **红色节点** — 故障节点

::: tip 流派选择
- **速攻流**：优先连接蓝色增伤节点，追求高爆发
- **控制流**：利用红色故障节点冻结或过热敌人
:::

**3. 武器系统**

游戏中有多种武器类型：
- 🔫 **射击武器**：冲击波枪、喷子、蓄力枪等
- ⚔️ **近战武器**：凿岩镐、大棒槌
- 🎯 **特殊武器**：月丝回收器、月丝加农炮

**4. 模组系统**

模组可装备到武器上提供各种加成：
- 诅咒、强硬进攻、超速充填、效能强化、热量转移、超速闪躲等

**5. 骇入节点**

用于增强骇入效果的芯片：
- 降防、冻结、过热、混乱、暴露

### 模式晶片

选择不同的模式晶片会改变战斗风格：
- **增幅模式**：增强骇入节点的伤害
- **充能模式**：加速技能冷却

## 狂月难度

狂月难度是游戏的最高难度，需要先通关一次普通难度后才能解锁。

::: warning 难度特点
- 敌人伤害和攻击速度提升
- 敌人血量明显增加
- 只能在一周目进行
- 无法携带升级物品进入
:::

详细打法请参考 [白金攻略](./白金攻略#狂月难度)。
