# 游玩感受

## 目前进度：月矿

- 能玩到这说明对这款游戏还是挺满意的
- 游戏开局指引做的还可以，战斗系统非常新颖，但是骇客的破解让我想起了蜘蛛侠
- 游戏平衡有点弱，普通的枪射击基本没用，需要搭配限制移动或者主动打倒敌人才能稳定命中要害，但是非PS5玩家可能可以打的很准？
- 我同时是PSP玩家，PSP玩这个破解机制真的不友好，PSP太重了呀
- 我是个仓鼠玩家，舍不得用消耗品，但是消耗品真的很强很强，尽量能用就用

## 白金向

- 难度不大，最高难度通关也很有挑战性，应该会是我的第38白金
- 提前看了白金攻略，没有可遗漏的，所以放心自行探索游玩，需要注意的是经常和小女孩对话积攒好感度
- 后期重复探索是容易迷路的，所以嫌麻烦的可以跟着攻略全收集
