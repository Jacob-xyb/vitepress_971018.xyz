# 《识质存在》模拟训练攻略

## 概述

模拟训练是《识质存在》中获取资源的重要渠道，总计 **30 个关卡**。完成训练可获得阿昌币、纯月矿、月石、升级组件、武器配件等核心养成资源。

### 解锁条件

1. 主线推进到**区域 2（大规模制造阵列）**
2. 回防护站与阿舱对话，解锁基础模拟训练入口
3. 后续训练靠收集地图里的**「训练数据」**（隐藏箱子、解谜、清精英怪）

### 训练类型

| 类型 | 核心内容 | 奖励 |
|------|----------|------|
| 限时清怪类 | 限时（90/100/120秒）清完所有敌人 | 阿昌币、纯月矿、月石、升级组件、武器配件 |
| 移动竞速类 | 限时抵达终点/收集目标点 | 阿昌币、推进器升级材料、纯月矿 |
| 骇入/破解类 | 一笔画骇入、破解敌人、不踩陷阱格 | 骇入组件、升级组件、阿昌币 |
| 生存/防御类 | 坚持时间、保护目标、抵御多波敌人 | 防御配件、月矿、升级组件 |

::: tip 推荐
前期优先清完所有**主目标**（限时清敌类），这是最稳定高效的资源获取渠道。次要目标不影响核心奖励，可随缘补。
:::

## 重点关卡攻略

### 关卡 10：捕鼠器 ⚠️ 易漏奖杯

**类型**：限时清怪

**特点**：怪物密集，适合刷「一网打尽」奖杯

**打法**：
1. 引怪聚团到开阔区域
2. 用月丝回收器对准群体释放静滞网
3. 一次性击倒三人即可解锁奖杯

### 关卡 16 ⚠️ 易漏奖杯

**「谁还需要枪？」** 可在此关刷出

**打法**：
1. 完成第二个阿舱集币板解锁「暴露」侵入节点
2. 关卡 16 有一地的暴露节点
3. 连接 5+ 个暴露节点可达成单次 3000 伤害

### 关卡 29：难度攻略

**敌人分四波出现**，节奏把控是关键：

1. 第一波：快速清理
2. 第二波：注意走位
3. 第三波：利用地形
4. 第四波：资源管理

::: tip 无伤通关技巧
虽然没有想象中那么难，但需要一波一波应对，不要贪刀。
:::

### 关卡 30：至死方休（最难）

**分波应对策略**：

1. **第一波**：捡起所有道具，放一个诱饵 + 两个入侵炸弹，一波带走
2. **第二波**：跳跳乐小心
3. **第三波**：等所有怪出来后，不要用任何道具。入侵掉一个打一点伤害（引诱治疗机器人），等所有怪都走到激光上直接激活秒杀
4. **第四波**：两把枪，穿 3 个，导弹 2 个，再穿 3 个，导弹三个
5. **第五波（最难）**：两拨敌人，做好资源管理
   - 先用诱饵和入侵炸弹用在三个步行者身上
   - 入侵连结混乱 3 个全部武器杀死
6. **第六波**：快冲快入侵，注意最后四个右边有一个是 QTE 的
