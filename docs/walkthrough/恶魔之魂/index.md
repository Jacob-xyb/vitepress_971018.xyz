# 恶魔之魂：重置版 攻略指南

## 游戏简介

《恶魔之魂：重置版》是由 Bluepoint Games 重制的动作角色扮演游戏，原作由 FromSoftware 开发。作为"魂"系列的开山之作，本作以高难度战斗、独特的死亡惩罚机制和黑暗奇幻世界观而闻名。重置版在保留原作精髓的同时，带来了次世代级别的画面表现和更加流畅的游戏体验。

## 游戏设定

### 核心机制

**1. 存档机制**

重制版的存档具有延迟特性，在人物死亡时直接关闭游戏可以回档。这一机制通常用于：
- 肉身状态推图时避免意外死亡
- 对付高难度 BOSS 时的保险手段

::: info
打白金时推荐灵魂状态推图因为容易死亡；肉身状态打Boss
:::

**2. 世界倾向系统**

世界倾向是影响奖杯获取的重要设定之一：

- **倾向等级**：
  - 世界倾向值 ±50% 及以上：白 +1 或黑 +1
  - 世界倾向值 ±75% 及以上：白 +2 或黑 +2
  - 世界倾向值 ±90% 及以上：纯白或纯黑

- **影响因素**：
  - 增加白值：击败 BOSS、原生恶魔、红灵 NPC
  - 增加黑值：肉身状态死亡、击杀 NPC
  - 联机影响：协助玩家击败 BOSS、红灵会影响倾向

- **倾向效果**：
  - 影响当前区域敌人的强弱
  - 白 +2 或黑 +2 及以上会触发特殊事件

::: tip 建议
一周目建议离线游玩，避免被入侵影响世界倾向。
:::

**3. 角色倾向系统**

角色倾向同样是奖杯获取的关键设定：

- **影响因素**：
  - 增加白值：击败红灵 NPC
  - 增加黑值：击杀 NPC
  - 联机影响：击杀入侵玩家或入侵击杀玩家

- **倾向效果**：
  - 影响灵魂状态下的最大生命值和攻击力上限
  - 纯白或纯黑状态可获取特殊戒指

**4. 装备负重**

- 负重 50% 及以上：翻滚变得缓慢
- 负重 100% 及以上：无法翻滚

**5. 道具负重**

限制携带道具的数量。重制版中可直接将道具转移至仓库，该限制已被大幅淡化。

**6. 保时捷精虫**

没击败一个关底boss，保时捷精虫就会刷新。

如果被跑掉了可以试试原地保存并退出然后重进试试。

## 白金攻略

完整的白金奖杯攻略指南，助你解锁全部成就。

::: tip 白金要点
- **多周目需求**：白金至少需要 2 周目，部分奖杯依赖不同的世界倾向
- **倾向系统**：白倾向与黑倾向会显著影响世界和角色，需提前规划路线
- **武器收集**：必须收集所有独特武器、法术和奇迹
- **NPC 任务线**：部分奖杯需要完成特定 NPC 的完整任务线，注意不要误杀
- **离线游玩**：建议一周目离线进行，避免倾向被入侵影响
:::
