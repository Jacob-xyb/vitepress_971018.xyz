# 恶魔之魂：重置版 白金攻略

::: tip 攻略说明
本攻略追求白金速通，按照世界白→世界黑→个人白→个人黑的顺序攻略，方法得当只需 1.5 周目即可白金。
:::

## 白金路线概览

完成白金需要至少 2 周目，建议按照以下路线进行：

- **一周目**：正常通关，收集大部分道具和武器
- **二周目**：完成倾向相关奖杯，收集剩余道具

## 攻略前必读

### 1. 黑白机制基础

恶魔之魂分为：世界的黑/白，以及个人的黑/白，四种状态，都与奖杯关系密切。

::: info 攻略顺序
按照世界白→世界黑→个人白→个人黑的顺序攻略
:::

关于黑白原理和效果，可以参考：[《恶魔之魂重制版》世界倾向机制讲解](https://www.gamersky.com/handbook/202011/1339278.shtml)

### 2. 角色建立

::: tip 推荐配置
- **职业**：王族开局（自带回蓝戒指、蓝上限头饰、法杖和灵魂箭，中期起飞，无需研究复杂的强化系统）
- **性别**：女（可穿加魂的银镯）
:::

::: warning 注意
不是法爷的同学，本攻略依然适用，但是火焰风暴实在是太强了。
:::

### 3. 流程目录

::: info 关卡说明
1-1 代表：世界1「柏雷塔尼亚王城」的第一个拱心石，下文同理
:::

**推荐流程顺序**：
1. 初探 1-1、1-2
2. 世界2打穿（2-1、2-2、2-3）
3. 救魔女（1-3）
4. 世界3打穿（3-1、3-2、3-3）
5. 世界4打穿（4-1、4-2、4-3）
6. 世界5打穿（5-1、5-2、5-3）
7. 收尾1-4 → 通关 → 二周目补完 → 白金

### 4. 推图注意事项

::: danger 重要提醒
1. 教学关死亡无所谓，会被剧情杀
2. 每张地图，都尽量以灵魂状态推图（除非每次都备份了存档）
3. 击杀 BOSS 之后要尽快回到神殿自杀，再进入关卡，不要在关卡内以肉身形态死亡
4. BOSS 魂别乱用
5. 一周目尽量地毯式探索，尤其是虚幻眼瞳石不要乱用，后续较缺
6. 一周目不要轻易击杀任何 NPC、不要轻易当红灵击杀玩家（影响个人纯白）
7. 养成经常检查黑白倾向的习惯
8. 不需要刻意清理保时捷精虫
:::

::: tip 法师蓝量管理
前期法师推图缺蓝，闲钱请在铁匠买新鲜香料，后面在 3-1 贵妇买陈年香料。
:::

### 5. 奇迹购买警告

::: danger 极其重要！
请务必不要在一周目用「龙神、黄衣老者、第六圣女」这三个魂买任何「奇迹」！

- 买魔法是可以的
- 想买奇迹请在二周目再买
- 否则铁定白金要三周目，而方法得当白金只要 1.5 周目即可

**请反复熟读并背诵这句话，关乎是否重开一个周目，很重要！**
:::

### 6. 周目继承机制

::: warning 存档提醒
进入新周目前请稳稳存档，新周目的世界倾向会继承，不刷新的！

- 一周目就尽量把能做的都趁着纯黑白都做了
- 这样二周目压力非常小，就是无脑通关即可
- 不然二周目还要重新刷黑白，痛苦至极
:::

### 7. 联机奖杯

::: tip 推荐做法
联机奖杯可以在最开始做，也可以在最后做。**推荐最后做**

- **推荐地点**：1-2 BOSS 房前
- **注意事项**：做之前存档招人，招人后再读档，避免打乱世界黑白值
:::

## 白金流程攻略

### 世界纯白阶段

#### 1-1

1. 必需道具：搜集【粘附戒指】、【盗贼戒指】、【翡翠发饰】（回城后给仓库管理员托马斯换取【巨力戒指】）
2. 重要道具：【巨剑】（2-3会用到）
3. 解救并保护王子（第一次，获得【黄铜望远镜】）
4. BOSS门口存档，击败1-1 BOSS 法兰克斯（BOSS魂作用会在文后统一注明） Boss怕火
5. 回家与火防女对话，在神殿二楼找到尊者，在谈话过程中尊者问你是否聆听愿望，一定要选择"yes"，不然一周目无法拿到"【盟友戒指】"
6. 找防火女升级，推1-2之前把魔法加到17，可以秒狗
7. 自杀，存档（自信就存档，肉身推图）

#### 1-2

1. 必需道具：搜集【耐伤戒指】
2. 重要道具：
    1. 【翼矛】（非常优秀的通关武器，想要体验近战的玩家可尝试此武器，配合持盾效果拔群）
    2. 【木制触媒】（伤害比银制触媒高，高周目合成发狂触媒需要一把）
    3. 【复合短弓】（合成熔岩弓需要一把，也是第一把弓箭）
3. 解救并保护王子（第二次）
4. BOSS门口存档，击败1-2 BOSS 高塔骑士，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀（不杀弓箭手的奖杯留到二周目）
5. 自杀，存档（自信就存档，肉身推图）

#### 2-1

1. 必需道具：搜集【耐毒戒指】、【官吏手套】、【官吏绑腿】（救魔女必需，2 世界有数个胖官吏掉落，数量绝对够，部分胖官吏不会掉落装备）
2. 重要道具：【克里斯匕首】（法爷的buff刀，可全程携带）、【粉碎战斧+1】
3. 开启通往黑铁匠铺的电梯
4. BOSS门口存档，击败2-1 BOSS 大蜘蛛，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀
5. 在家里自杀、检查本小节是否有遗漏、线上存档

#### 2-2

1. 必需道具：【耐疾戒指】（捷径中途洞穴对面的木架上）、【耐炎戒指】、【黑瞳石】（水晶蜥蜴窝前的两个红灵掉）、【官吏服】、【天神之手x2】
2. 重要道具：【龙之长剑+1】、【挖洞者套装】（前期最高火抗）、【卓越斧】、【遮光热盾+2】（前期神盾）、【棍棒】（合成切肉刀需要一把）、【纯粹硬石】和【纯粹锐石】（水晶蜥蜴窝捡，顶级素材）
3. 从火甲虫那里触发奸商佩奇（不管他也行，可以留到4-2速救）
4. BOSS门口存档，击败2-2 BOSS 火焰潜伏者，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀
5. 把【2-2BOSS赤热的恶魔灵魂】给2-1铁匠的，开启高级装备合成，跳杯
6. 回家升级，推荐力量加到12，可以用巨剑、盾、弓
7. 在家里自杀、检查本小节是否有遗漏、线上存档

#### 2-3

1. 必需道具：【大师戒指】
2. 重要道具：【纯粹龙石】、【碎龙骨】
3. BOSS门口存档，正常来说2世界此时已经纯白，没有纯白就再肉身击杀龙神
4. 击败2-3 BOSS龙神，推荐使用【巨剑】清障碍【天神之手】的成就可以留到二周目做（但是其实装在副手上击杀也跳杯）
5. 回家，帕奇处买【芳香戒指】（王族开局自带）
6. 升级，尽快把智力增加到20点以上
7. 在家里自杀、检查本小节是否有遗漏、线上存档

#### 1-3

救魔女学火焰风暴

1. 必需道具：【官吏帽】、【铁钥匙串】、【沾血的铁钥匙】（都是救魔女用的）、【魔法天性戒指】、【被诅咒者戒指】
2. 重要道具：搜集【卓越剑】、【塔盾】（可以忽略，1-4流程必掉）
3. 找到1-3的奴隶兵商人
4. 解救并保护王子（第三次），与王子对话获得【纯粹弥漫石】
5. 解救比约尔
6. 装备官吏套装，解救魔女，（跟魔女对话要脱下来，不然她不会回家）
7. 回家升级智力，增加法术空格，找魔女学习火焰风暴，秒BOSS的神技
8. 在家里自杀、检查本小节是否有遗漏、线上存档

#### 1-3 打穿

1. 稳扎稳打，推到1-3 BOSS门口，三只红眼枪兵弓箭勾引，逐个击杀
2. 在BOSS门口存档，与比约尔一同击败3-3BOSS 贯穿骑士，火焰风暴不要把比约尔打死了
3. 在BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀，此时1世界已经纯白
4. 回家升级、在家里自杀、检查本小节是否有遗漏、线上存档

#### 3-1

1. 必须道具：搜集【魔法锐性戒指】（推荐法师全程佩戴）、【巧鼠戒指】
2. 重要道具：【法师套装】、【木制触媒】、【白银冠冕】、【漆黑套装】（毒抗很高）、【暗杀者面罩】、【钩爪】、【隐秘匕首】（背刺3-3用）、【重型弓弩】、【沟纹套装】（性价比很高的一套）、【鸢盾】、【银手镯】（加魂，推荐全程佩戴）、【穿甲剑】、【水银刺剑+1】（升级水银武器首选，免去之前的基底）、【银质触媒】、【王国密探套装】、【格里的细剑】、【婴孩厉爪】（杀掉BOSS之后不要靠近祭坛，贴墙边捡取）
3. 找到贵妇商人，购买陈年香料（再多也不嫌多，法师很缺蓝）和【贪欲戒指】（钱不够就等3-2拣）
4. 在BOSS箭雨大桥前存档，尝试翻滚过箭雨大桥，听到声音翻滚两下，跳杯
5. 解救法师弗瑞克（钥匙在BOSS楼上的奴仆处，把奴仆杀掉）
6. BOSS门口存档，击败3-1 BOSS愚者偶像，通过锁定看血条和技能大小来找出本体
7. 注意不要伤害到分身，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀，跳特殊杯
8. 回家升级、在弗瑞克购买法术【灵魂射线】、在家里自杀、检查本小节是否有遗漏、线上存档

#### 3-2

1. 必需道具：无（不追求全收集可以全部忽略）
2. 重要道具：搜集【焰形剑】、【馨香戒指】（第三枚，贵族自带、帕奇购买）、【耐疾戒指】（2-2入了一枚）、【贪欲戒指】（3-1贵妇可以买）、【黄金面具】、【锐刺剑】（暗杀任务最后杀死梅菲斯特也会掉落）、【符文剑】＆【符文盾】（1-4也会获得）、【月光长剑+1】（吃人鬼BOSS房里）
3. 解救并当场杀死沉默者约尔特，获得【幽暗套装】（非常重要，杀他可以开启后续的个人纯黑任务，或者回神殿第一时间杀他也可，都不影响世界和个人黑白，不然他会疯狂杀家里的NPC）
4. BOSS门口存档，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀
5. 击败3-2 BOSS吃人鬼，击杀后3世界达到纯白
6. 回家升级、强化【隐秘匕首】、在家里自杀、检查本小节是否有遗漏、线上存档

#### 3-3

1. 推到BOSS门口，断网，存档
2. 击败3-3 BOSS黄衣老者，没纯白就继续捏石头击杀，用【隐秘匕首】背刺很好抓

#### 4-1

1. 必需道具：搜集【铜制钥匙】、【掘墓者戒指】、【再生者戒指】
2. 重要道具：【复合长弓】、【打刀】、【灰色恶魔之魂】（击杀恶魔先锋）、【新月圆月刀+1】（法师用起来还不错）、【月影石的碎片】、【薄云石】、【刃石】、【审判者之盾】
3. 解放商店NPC——掘墓者布莱吉商人
4. BOSS门口存档，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀
5. 击败4-1 BOSS审判者，在二层远程biu死，跳特殊击杀奖杯
6. 回家升级、在家里自杀、检查本小节是否有遗漏、线上存档

::: tip 捷径提示
去BOSS有一个捷径：有两个弓箭手的那段城墙，走到尽头是个向下的楼梯，不要下去，对角线朝着墙面的缺口加速跑，感觉踩到墙头的瞬间按翻滚，会翻过城墙掉到，拾取再生者戒指的位置（二周目跑酷专用）
:::

#### 4-2

1. 必需道具：【浪人戒指】（在蛞蝓水池内，击杀世界纯白下2-2的龙骨碎痴迷者同样可掉落，推荐这里和无柄一起捡了，不推荐杀NPC）
2. 重要道具：【无柄】（若遗漏无须回头再找，世界纯黑4-1上来就杀过来的红灵NPC必掉落）、【白弓】（1-4流程必掉可以不捡，在4-2拱心石出来，按流程跑到有死神召唤虚影怪的桥上，直接从左手边跳，上2楼转角处右手边有一扇堵死的、可以看出轮廓的门，砍一刀可以开启隐藏墙，击杀双刀骷髅，走到底可以搜集到，）、【耐伤戒指】（帕奇给的，1-2已入一枚）、【匕首】、【纯粹暗夜石】、可能会获得【纯粹刃石】（怨念物）
3. 触发帕奇剧情，救出圣职于尔班
4. BOSS门口存档，BOSS丝血的时候使用虚幻眼瞳石，恢复肉身击杀
5. 击败4-2 BOSS 旧日勇士，此时世界4纯白
6. 回家升级、在圣者处用老勇士的魂兑换奇迹【仅此一次的复活】（非常好用的奇迹，只要蓝够可以无限挂，推图多一条命，尽早升级奇迹装上，其他奇迹不要买，否则白金需要三周目）
7. 在家里自杀、检查本小节是否有遗漏、线上存档

::: warning 注意
4-2堆怪之严重随时让人心态崩盘，建议随时存档

5. 世界5迅速打穿

至此，所需要的BOSS魂就集齐了，请务必存档！！

::: tip 白金达成
- 先在于尔班处补全缺失的奇迹，全奇迹跳杯，回档
- 然后在弗瑞克处补全缺失的魔法，全魔法跳杯！

**【白金达成】！！**
:::

白金后可以继续回档，选择喜欢的奇迹和魔法购买，或者合成BOSS魂武器也可以（疯狂触媒伤害爆炸）

### 贯穿骑士套补完（可选）

如果白金后你还有余力，可以继续收集二周目纯黑纯白世界下的13个陶币

攒齐26个陶币后，不用着急去兑换，可以分享给朋友，让世界充满爱

## 白金技巧

### 倾向控制

**世界倾向控制**：
- 增加白值：击败 BOSS、原生恶魔、红灵 NPC
- 增加黑值：肉身状态死亡、击杀 NPC
- 建议：一周目保持纯白，二周目刷纯黑

**角色倾向控制**：
- 增加白值：击败红灵 NPC
- 增加黑值：击杀 NPC
- 建议：先刷纯白，再刷纯黑（纯黑更容易）

### 存档技巧

**回档方法**：
1. 在关键节点手动上传存档到云端
2. 如果出错，下载云存档覆盖本地存档
3. 或在死亡瞬间关闭游戏（利用延迟存档机制）

**应用场景**：
- 避免肉身状态意外死亡
- 避免误杀 NPC
- 倾向刷错时回档

## 常见问题

### Q: 白金需要多长时间？
A: 根据熟练度不同，大约需要 40-60 小时。

### Q: 必须联机吗？
A: 不需要，建议离线游玩以更好地控制倾向。

### Q: 错过了某个道具怎么办？
A: 可以在二周目或后续周目获取，大部分道具都可以重复获得。

### Q: 倾向刷错了怎么办？
A: 可以通过回档或继续游戏联机调整倾向，倾向是可以反复变化的。

### Q: NPC 被误杀了怎么办？
A: 如果有云存档可以回档，否则只能等下一周目。
