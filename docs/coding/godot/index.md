# Godot

开源、免费、轻量级的游戏引擎。

## 下载

### 官方下载地址

- **官网**: [godotengine.org/download](https://godotengine.org/download)
- **Steam**: 搜索 "Godot"（集成Steam版）

### 版本选择

| 版本 | 说明 |
|------|------|
| Godot 4.x | 当前稳定版，功能丰富，推荐使用 |
| Godot 3.x | 经典版，部分项目仍在使用 |

### 各平台版本

- **Windows**: `.exe` 安装包 / `.zip` 便携版
- **Linux**: `.x86_64.zip` 便携版，部分发行版可通过包管理器安装
- **macOS**: `.dmg` 安装包

### Mono 版本

如需使用 C# 脚本，需下载带 **.NET** 标识的版本（内置 Mono/IL2CPP 运行时）。

### 版本对比

| 版本 | 脚本 | 适用人群 |
|------|------|----------|
| 普通版 | GDScript | 初学者、纯独立游戏开发 |
| .NET 版 | C# | 熟悉 C#、需要接 .NET 库 |

### 推荐下载

去 [godotengine.org/download](https://godotengine.org/download) 下载：

- **版本**: Godot 4.x（当前主流）
- **平台**: Windows 64-bit（普通版，不是 .NET）

::: tip 推荐理由
- **教程多** - 95% 以上的 Godot 教程用 GDScript
- **更轻量** - 下载小，无需安装 .NET SDK
- **集成更好** - GDScript 是 Godot 原生语言，编辑器支持更完整
- **够用** - 做独立游戏不需要 C# 的性能或生态
:::
