# Go 编程

Go 是一门简洁、高效、可靠的服务器端编程语言，由 Google 开发维护。

## 为什么选择 Go

Go 语言设计初衷是解决 Google 内部的大规模服务端开发问题，兼具 Python 的简洁和 C++ 的性能。

**核心优势**

- 简洁易学，语法清晰，上手快
- 编译速度快，部署简单
- 优秀的并发支持（goroutine + channel）
- 垃圾回收，无需手动内存管理
- 丰富的标准库和工具链
- 静态编译，单二进制文件部署

**适用场景**

- 服务器端开发 / 网络编程
- 微服务架构
- 云原生应用（Kubernetes、Docker 等）
- 命令行工具（CLI）
- 数据处理和管道
