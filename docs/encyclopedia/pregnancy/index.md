# 孕期常识

孕期健康与护理知识汇总，帮助准妈妈度过健康快乐的孕期。

## 孕期阶段划分

### 孕早期（1-12周）
- 胚胎着床，器官开始形成
- 容易出现孕吐、疲劳等早孕反应
- 关键时期，需要特别注意避免接触有害物质
- 建议补充叶酸（400-800微克/天）

### 孕中期（13-27周）
- 早孕反应逐渐消失，食欲恢复
- 胎儿快速生长发育
- 开始感受到胎动（16-20周左右）
- 相对稳定舒适的时期

### 孕晚期（28-40周）
- 胎儿进入快速增重期
- 可能出现水肿、腰酸背痛
- 需要注意胎动监测
- 准备待产包，学习分娩知识

## 营养补充

### 必需营养素
- **叶酸**：孕早期每天400-800微克，预防神经管畸形
- **钙**：每天1000-1200毫克，促进胎儿骨骼发育
- **铁**：每天20-30毫克，预防贫血
- **DHA**：每天200-300毫克，促进胎儿大脑发育
- **维生素D**：每天10微克，帮助钙吸收

### 饮食原则
- 少食多餐，避免空腹和过饱
- 多吃新鲜蔬菜水果
- 适量摄入优质蛋白（鱼、肉、蛋、奶、豆制品）
- 保证充足水分摄入
- 避免生冷、辛辣刺激食物

### 禁忌食物
- ❌ 生鱼片、生肉、生蛋
- ❌ 未经巴氏消毒的奶制品
- ❌ 酒精饮料
- ❌ 高汞鱼类（鲨鱼、旗鱼、金枪鱼）
- ❌ 过量咖啡因（每天不超过200mg）

### 咖啡每天可以喝多少？

之所以一般不建议孕期喝咖啡，或者建议严格限制，核心原因在于咖啡因这种物质。对于孕妈妈的身体和发育中的胎儿来说，咖啡因的影响比平时要复杂得多。

主要有以下几个关键原因：

胎儿代谢能力弱
咖啡因可以通过胎盘进入胎儿体内。但与成人不同，胎儿的肝脏和肾脏功能尚未发育成熟，分解和代谢咖啡因的酶系统活性很低。这意味着，咖啡因会在宝宝体内停留的时间比在妈妈体内长得多，可能对宝宝产生更持久的刺激。

增加流产和低出生体重风险
这是医生们最谨慎的原因。大量的流行病学研究显示，高剂量的咖啡因摄入（通常指每天超过200-300毫克）与流产风险的轻微增加，以及新生儿出生体重偏低有一定关联。虽然这个风险并非绝对，但为了安全起见，医学界倾向于采取保守态度。

影响母体对铁的吸收
咖啡中的单宁酸会干扰身体对非血红素铁（主要来自植物和铁补充剂）的吸收。孕期本身就容易出现缺铁性贫血，如果在饭后马上喝咖啡，可能会让这个问题雪上加霜，影响妈妈的健康和宝宝的氧气供应。

加重孕期不适
怀孕后，很多妈妈会对各种物质变得更敏感。咖啡因的利尿作用可能会加重尿频；它也可能刺激胃酸分泌，让本就容易出现的烧心、胃食管反流变得更加难受；同时，它还可能影响睡眠质量，让孕晚期的准妈妈更加疲惫。

那么，孕期就完全不能碰咖啡了吗？
其实也不是绝对的。目前国际上主流的观点（如美国妇产科医师学会ACOG）认为，孕期适量摄入咖啡因是安全的。这个“适量”通常指的是每天不超过200毫克。

为了让你对这个剂量有个直观的概念，这里有一些参考：

现煮咖啡：一杯（约240毫升）大约含有 95-165毫克 咖啡因。

速溶咖啡：一杯（约240毫升）大约含有 60-80毫克 咖啡因。

拿铁/卡布奇诺：如果只含一份浓缩咖啡，咖啡因含量约 75毫克。

红茶：一杯（约240毫升）大约含有 40-50毫克 咖啡因。

可乐/能量饮料：需要留意，这些饮料也含有咖啡因，而且糖分很高。

所以，如果你很想喝，偶尔喝一小杯拿铁或速溶咖啡，通常是在安全范围内的。关键在于“总量控制”，不仅要算咖啡，还要把当天喝的可乐、浓茶、吃的高纯度黑巧克力等所有含咖啡因的食物都算进去。

## 产检时间表

| 孕周 | 检查项目 | 重要性 |
|------|---------|--------|
| 6-8周 | 确认宫内孕、胎心 | ⭐⭐⭐ |
| 11-13周 | NT检查（早期唐筛） | ⭐⭐⭐ |
| 15-20周 | 中期唐筛/无创DNA | ⭐⭐⭐ |
| 20-24周 | 大排畸（四维彩超） | ⭐⭐⭐ |
| 24-28周 | 糖耐量测试（OGTT） | ⭐⭐⭐ |
| 28周后 | 每2周产检一次 | ⭐⭐ |
| 36周后 | 每周产检一次 | ⭐⭐⭐ |

## 常见不适及应对

### 孕吐
- 早晨起床前吃点饼干或面包
- 少食多餐，避免空腹
- 避免油腻、辛辣食物
- 严重时及时就医

### 便秘
- 多吃富含纤维的食物
- 保证充足水分
- 适量运动
- 必要时咨询医生使用开塞露

### 水肿
- 避免久站久坐
- 睡觉时抬高双腿
- 控制盐分摄入
- 左侧卧位休息

### 腰酸背痛
- 保持正确姿势
- 使用孕妇枕
- 适度运动（孕妇瑜伽、散步）
- 避免提重物

## 生活注意事项

### 运动
- ✅ 推荐：散步、孕妇瑜伽、游泳
- ⚠️ 避免：剧烈运动、跳跃、仰卧起坐
- 每天30分钟中等强度运动
- 运动时注意补水

### 睡眠
- 孕中晚期建议左侧卧位
- 使用孕妇枕辅助
- 保证每天8-9小时睡眠
- 睡前避免大量饮水

### 工作
- 避免长时间站立或久坐
- 远离辐射、化学物质
- 适时休息，不要过度劳累
- 孕晚期可考虑提前休假

### 出行
- 孕中期（13-27周）是相对安全的出行时期
- 长途旅行需咨询医生
- 乘车系好安全带（腹部下方）
- 避免去人多拥挤的地方

## 危险信号（需立即就医）

- 🚨 阴道出血或流液
- 🚨 剧烈腹痛
- 🚨 持续头痛、视力模糊
- 🚨 胎动明显减少或消失
- 🚨 严重水肿（手、脸部）
- 🚨 高烧不退
- 🚨 持续呕吐无法进食

## 心理健康

### 情绪管理
- 激素变化可能导致情绪波动
- 与家人朋友多沟通
- 保持积极乐观心态
- 必要时寻求专业心理咨询

### 产前准备
- 参加孕妇学校或产前课程
- 了解分娩过程和方式
- 准备待产包
- 与伴侣一起学习育儿知识

## 实用工具

### 预产期计算
预产期 = 末次月经第一天 + 280天（40周）
简便算法：末次月经月份 + 9（或 - 3），日期 + 7

### 体重增长参考
- 孕前BMI正常（18.5-24.9）：增重11-16kg
- 孕前偏瘦（BMI<18.5）：增重12.5-18kg
- 孕前超重（BMI 25-29.9）：增重7-11.5kg
- 孕前肥胖（BMI≥30）：增重5-9kg
