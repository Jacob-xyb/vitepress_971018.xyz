# 孕期事务

孕期需要办理的手续一览，帮你理清哪些需要办、哪些不需要。

## 已取消事项 ❌

| 事项 | 说明 |
|------|------|
| ~~准生证~~ | 国家已全面取消，实行生育登记服务制度 |
| ~~生育医疗登记~~ | 武汉市 2022 年 10 月起取消，可直接持医保卡结算 |

::: tip 注意
男职工未就业的配偶，仍需办理生育就医登记才能享受男方生育险待遇。
:::

## 概念区分 📋

| 概念 | 性质 | 核心作用 |
|------|------|----------|
| **生育登记** | 行政登记手续 | 备案证明，为落户、享受公共服务提供依据 |
| **社区建档** | 健康管理服务 | 领取《母子健康手册》，记录健康信息 |

::: info 小提示
武汉市绝大多数医院已不强制要求生育登记证明建档。少数情况下，它可作为福利申领依据（如免费无创 DNA）。
:::

## 生育登记 📝

### 办理时间

确认怀孕后即可办理，生产前办好即可，时间充裕。

### 线上办理（推荐）✅

登录 **"鄂汇办"APP** 或 支付宝搜索 **"鄂汇办"小程序**，搜索"生育登记"。

### 线下办理

| 材料 | 说明 |
|------|------|
| 夫妻双方身份证 | 原件 |
| 户口簿 | 原件 |
| 结婚证 | 原件 |

办理地点：一方户籍地或现居住地的乡镇(街道)便民服务中心、村(居)委会卫健窗口。

### 新生儿"一件事" 👶

孩子出生后，通过"鄂汇办"APP 一站式办理：

- 出生证明
- 落户
- 新生儿医保
- 社保卡

::: info 提醒
《母子健康手册》属于医疗健康管理范畴，需尽早去医院办理，与行政手续不冲突。
:::

## 社区建档 🏥

这是孕期健康管理的起点，领取《母子健康手册》并享受国家公共卫生福利。

| 项目 | 内容 |
|------|------|
| **办理地点** | 常住地所在的社区卫生服务中心（注意是医疗机构，不是居委会） |
| **办理时间** | 孕 6 周后至 12 周前 |
| **办理材料** | 夫妻双方身份证 + 证明怀孕的 B 超单/血尿检报告 |

### 所做事项

- 领取《母子健康手册》—— 每次产检务必带上
- 完成基础检查（量血压、听胎心等）
- 了解国家提供的免费产检项目（如艾滋病、梅毒筛查）

## 医院建档 🏩

拥有《母子健康手册》后，在目标分娩医院建立更专属的档案。

| 项目 | 内容 |
|------|------|
| **办理地点** | 计划产检和分娩的医院产科 |
| **办理时间** | 孕 12 周左右（建议提前 1-2 周预约） |
| **办理材料** | 《母子健康手册》、夫妻双方身份证、医保卡、已完成的检查报告、生育登记信息 |

::: warning 不同医院要求不同
像市妇幼建议孕 8-12 周完成预约和建档，号源紧张，请提前在医院公众号预约。
:::

### 所做事项

- 全面的身体检查（身高、体重、妇科检查、抽血、心电图等）
- 医生详细询问末次月经、孕产史、用药史等
- 了解后续产检安排

## 大排畸（NT 检查）🔍

| 项目 | 内容 |
|------|------|
| **最佳时间** | 孕 11 周 至 13 周 + 6 天 |
| **建议** | 可与医院建档同时进行，更省心 |

::: tip 小贴士
孕 12 周去医院建档时，可以把 NT 检查一起安排，一举两得。
:::
