# 孕期事务

## 已取消事项

- 准生证

国家层面已全面取消准生证制度‌，实行生育登记服务制度，无论生育几个孩子，均无需再办理准生证。

- 生育医疗登记

武汉市自**2022年10月起取消了“生育就医登记”‌**，即女职工（包括灵活就业和城乡居民医保参保人员）在生育分娩、流引产或计生手术前，‌不再需要专门办理生育就医登记‌，可直接持医保电子凭证或社保卡在定点医疗机构即时结算相关费用。

但是 **男职工未就业的配偶** 是需要办理的，让你能享受男方的生育险待遇。

## 概念区分

**生育登记**：是一项行政登记手续。核心作用是“备案”，证明你的生育符合国家政策，为后续办理新生儿落户、享受部分公共服务（如免费筛查、部分补贴）提供依据

**社区建档**：是一项健康管理服务。主要目的是在社区卫生服务中心领取你的《母子健康手册》，记录你和宝宝的健康信息，是后续所有产检的基础。

武汉市绝大多数医院和社区卫生服务中心已不再强制要求提供此证明用于建档。

唯一的变数是，少数医院或特殊情况下，提供它可能被视为一项福利申领依据，或是某些特定检查（如免费无创DNA）的补充条件。

## 生育登记

办理时间：确认怀孕后即可办理，生产前办好就行，时间非常充裕。

是新生儿落户、领取生育服务手册和部分补贴的依据。它现在是一个服务性登记，不是行政审批，非常方便。你可以：

**线上办理 (推荐)**：登录 “鄂汇办”APP 或支付宝搜索 “鄂汇办”小程序 或 “湖北政务服务网”，搜索“生育登记”。

线下办理：带上夫妻双方的 《身份证》、《户口簿》、《结婚证》 原件，到一方户籍地或现居住地的乡镇(街道)便民服务中心、村(居)委会卫健窗口办理。

---

**生完宝宝后：办理新生儿“一件事”**

孩子出生后，还可以通过“鄂汇办”APP上的新生儿出生“一件事”服务，一站式办理出生证明、落户、新生儿医保、社保卡等所有后续手续。

另外想提醒你，《母子健康手册》(俗称“建档”)属于医疗健康管理范畴，也需要尽早去医院办理，但和上述行政手续不冲突。

## 社区建档

这是孕期健康管理的起点，作用是领取《母子健康手册》并享受国家公共卫生福利。

办理地点：你常住地所在的社区卫生服务中心。

	注意：是带“医院”或“中心” 字样的医疗机构，不是社区居委会。

办理时间：孕6周后至12周前。为方便，建议和社区确认一下具体工作时间。

办理材料：带上夫妻双方的身份证和能证明你怀孕的B超单（或血/尿检报告）就行。

所做事项：

领取最重要的《母子健康手册》，相当于宝宝和你的健康档案，每次产检都务必带上。

完成一些基础检查，如量血压、听胎心等。

了解并享受国家提供的免费产检项目（如部分地区的艾滋病、梅毒筛查等）

## 医院建档

医院建档：建“档案” 📂
拥有《母子健康手册》后，在目标分娩医院建立一个更专属的档案，方便医生系统管理。

办理地点：你计划产检和分娩的目标医院产科。

办理时间：完成社区建档后，孕12周左右。注意，不同医院对建档的孕周要求可能不同，像市妇幼就建议在孕8-12周完成预约和建档。考虑到号源紧张，强烈建议提前1-2周在医院公众号上预约建档号。

办理材料：带上《母子健康手册》、夫妻双方身份证、医保卡、已完成的检查报告及生育登记信息。

所做事项：

- 进行一次全面的身体检查，包括身高、体重、妇科检查、抽血、心电图等。

- 医生会详细询问你的末次月经、孕产史、用药史等，并网上录入你的完整信息，正式建立档案。

- 医生会告知你后续的产检安排。

## 大排畸

NT检查：这是孕早期最重要的一次排畸检查。最佳检查时间是孕11周至13周+6天。所以，在孕12周去社区或医院办理医院建档时，就可以把这个检查和产检安排在一起，会更省心。
